Composer instructions
===========================

## Инструкция
### Добавить библиотеку в composer
```json
{
    "require": {
        "theiconic/php-ga-measurement-protocol": "^2.0"
    }
}
```

### Обновить и загрузить библиотеку в vendor
Выполнить команду:
```php
composer update "theiconic/php-ga-measurement-protocol": "^2.0"
```
